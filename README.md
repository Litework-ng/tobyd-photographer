# TobyDphotographer — Official Website

Artist website for **Omoruyi Eghosa** (TobyDphotographer), Lagos-based self-portrait photographer working at the intersection of photography and Renaissance painting.

## Stack
- Pure HTML/CSS/JS — zero dependencies, zero build step
- Deployed on Netlify via GitHub auto-deploy

## Deployment
Push to `main` branch → Netlify auto-deploys in ~30 seconds.

## Structure
```
index.html      ← entire site (single file)
netlify.toml    ← Netlify config + cache headers
README.md       ← this file
```

## To update the site
1. Edit `index.html`
2. `git add . && git commit -m "your message" && git push`
3. Netlify picks it up automatically
